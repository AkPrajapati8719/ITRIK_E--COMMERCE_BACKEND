## 🔑 AUTHENTICATION

### How to Login:
1. Open http://127.0.0.1:8000/index.html
2. Click "Login" button (top right)
3. Enter **Full Name**
4. Enter **10-digit Mobile Number** (e.g., 9876543210)
5. Click "Send OTP"
6. Check console or terminal for OTP (printed for testing)
7. Enter OTP (4 digits)
8. Click "Verify & Login"

### First Time Users:
- System will ask to complete profile
- Enter: Email, Mobile, Address, City, Pincode
- Click "Complete Profile"
- Now you can shop!

### Test Credentials:
```
Mobile: 9876543210 (or any 10-digit number)
OTP: Will be printed in console/terminal
```

## 📱 API ENDPOINTS (For Developers)

### Base URL
```
http://127.0.0.1:8000/api
```

### Authentication
```
POST /auth/send-otp/
POST /auth/verify-otp/
GET  /auth/profile/
PUT  /auth/profile/
POST /auth/profile/complete/
```

### Products
```
GET  /products/                    - List all products with filters
GET  /products/{id}/               - Get product details
GET  /categories/                  - List categories
POST /products/{id}/add_review/    - Add review to product
```

### Shopping Cart
```
GET    /cart/                      - Get user's cart
POST   /cart/add/                  - Add item to cart
DELETE /cart/remove/               - Remove item
PATCH  /cart/update/               - Update quantity
DELETE /cart/clear/                - Clear cart
```

### Orders
```
GET    /orders/                    - List user's orders
POST   /orders/                    - Create order
GET    /orders/{id}/               - Order details
POST   /orders/{id}/cancel/        - Cancel order
GET    /orders/{id}/estimate_delivery/ - Estimated delivery
GET    /orders/{id}/track/         - Track order
```

### Content
```
GET  /blog/                        - List blog posts
POST /blog/                        - Create blog (admin)
PUT  /blog/{id}/                   - Edit blog (admin)

GET  /contact/                     - List messages (admin)
POST /contact/                     - Submit contact form
POST /contact/{id}/mark_read/      - Mark read (admin)

GET /settings/                     - Get site settings
PUT /settings/                     - Update settings (admin)
```

## 🔐 SECURITY

### Protected Endpoints
- All `/cart/` endpoints require login
- All `/orders/` endpoints require login
- Blog create/edit/delete require admin access
- Contact messages require admin access
- Site settings require admin access

### Token Usage
```javascript
// Example: Add to cart
fetch('http://127.0.0.1:8000/api/cart/add/', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({
    product_id: 'product-uuid',
    quantity: 1
  })
})
```

## 🆘 TROUBLESHOOTING

### Server Won't Start
```bash
cd itrik_backend
python manage.py runserver
```

### Migrations Issue
```bash
cd itrik_backend
python manage.py migrate
```

### Admin Access Denied
```bash
# Create superuser
cd itrik_backend
python manage.py createsuperuser
```

### CORS Error
- Make sure server is running on http://127.0.0.1:8000/
- Check settings.py CORS_ALLOWED_ORIGINS

### Products Not Showing
- Check if products exist in database
- Visit http://127.0.0.1:8000/api/products/
- Should see JSON list of products

### Cart Not Working
- Make sure you're logged in
- Check Authorization header has valid token
- Clear browser cache
